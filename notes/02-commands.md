# 指令

```bash
# 建立名為 my-app 的專案
npx create-expo-app my-app

# 列出範例專案的清單
npx create-expo-app --template
```

```bash
# 安裝網站所需套件
npx expo install react-dom react-native-web @expo/metro-runtime
```

```bash
# 啟動開發用伺服器
npx expo start

# 列出 Expo CLI 的可用指令清單
npx expo --help
```
